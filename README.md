## ✨ Features

### 🧠 AI-Powered Accountability
*   **The "Why" Interrogation:** You cannot simply swipe away missed tasks. When a deadline passes, the built-in Gemini AI intercepts it. You must explain *why* you missed it, and the AI will negotiate and reschedule the task to a new preferred date and time.
*   **Sincerity Insights:** Get a one-click analysis of your daily and all-time discipline, scoring your timeliness and task completion rates.
*   **Monthly AI Battles:** Select multiple months in your profile to generate a comparative behavioral report. The AI analyzes your positive and negative habits and ranks your selected months based on consistency.

### 📋 Core Task Management
*   **Zero-Excuses Exact Deadlines:** Powered by Android `AlarmManager` and `BroadcastReceiver` to ensure pinpoint accuracy for reminders and missed-task detection.
*   **Daily Rollover System:** Unfinished tasks automatically roll over, forcing you to confront procrastination.
*   **Premium Dark-Mode UI:** A minimalist, card-based Material Design 3 interface that is fast, fluid, and distraction-free.
*   **100% Offline-First:** Powered by Room Database. Your tasks and local data stay entirely on your device.

### 👤 Personalization
*   **Custom Profiles:** Set your display name and personalize your space.
*   **Premium Avatars:** Upload your own photo or choose from a curated selection of pre-filled cartoon animal avatars (inspired by Google Play Games / Prime Video profiles).

## 🚀 Getting Started

If you want to clone the repository, explore the code, or build it yourself, follow these steps:

### 1. Clone the repository
```bash
git clone [https://github.com/yourusername/your-repo-name.git](https://github.com/yourusername/your-repo-name.git)

```

### 2. Open in Android Studio

Open Android Studio, select **Open an existing project**, and navigate to the cloned directory.

### 3. Add your Gemini API Key

To make the AI features work locally, you will need a Google Gemini API key.

1. Get an API key from Google AI Studio.
2. Open the `local.properties` file in the root of your Android Studio project.
3. Add the following line:

```properties
GEMINI_API_KEY="your_actual_api_key_here"

```

*(Ensure your `local.properties` is included in your `.gitignore` to keep your key secure!)*

### 4. Build and Run

Sync the Gradle project and hit **Run** to deploy it to your emulator or physical device.
